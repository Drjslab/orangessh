## 🚀 Overview

OrangeSSH is a modern Android SSH client designed for DevOps engineers, Linux administrators, cloud engineers, and developers who need secure remote access on the go.

## ✨ Features

- Secure SSH Terminal
- Identity Management
- Encrypted Backups
- Command Snippets
- AI Assistant
- AES-256 Encryption
